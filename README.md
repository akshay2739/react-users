### Install the dependencies using `npm install` in root directory

### To run the application `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Notes

1. Add the Google Maps API key in AddressDetails components (./src/components/AddressDetails.js) on line number 50.
2. As Google maps weird behaviour page has horizontal and vertical scrolling.
